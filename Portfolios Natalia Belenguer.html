PROYECTO DE AUTOMATIZACIÓN INDUSTRIAL DE UN SISTEMA DE CLIMATIZACIÓN HVAC

Trabajo final de grado

Fecha: 01/03/2020

Accede:

El trabajo final de grado consiste en la automatización industrial de un sistema de climatización HVAC para la realización de una sala limpia clasificada. Para la ejecución de este proyecto he utilizado un autómata programable de la marca Siemens S7-1500 con Software de programación STEP7(TIA Portal) y para el interfaz visual con el usuario, he diseñado el HMI y el servidor web.
